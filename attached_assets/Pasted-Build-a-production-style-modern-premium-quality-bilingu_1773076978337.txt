Build a production-style, modern, premium-quality bilingual web app called "StudyFlow AI" for students in Bangladesh first, with full English and Bangla support across the interface and inside the AI assistant.

Primary goal:
Create an AI-integrated academic study web app with:
1. study planning and scheduling,
2. focus tools like Pomodoro,
3. progress tracking,
4. a professional academic AI chatbot,
5. friend system and direct messaging,
6. public status and private quick notes,
7. local-first AI chat history saved on the user's device,
8. managed email login without requiring a separately deployed custom backend server.

Important architecture constraints:
- This must be a web app, optimized for mobile and desktop.
- Make it a PWA, installable on phones.
- Use a single Replit project.
- Do not require a separately deployed custom backend server beyond the app itself.
- It is acceptable to use managed services such as Supabase or Firebase for auth, presence, friends, and messaging.
- AI assistant chat history must be stored locally on the device by default using IndexedDB or another reliable browser storage solution.
- User-to-user messaging must support shared cloud delivery using a managed backend because real messaging requires shared storage.
- Cache friend messages locally as well for fast mobile experience.
- Add encrypted export/import backup for AI chat history, so users can download a backup file and optionally share it through email manually.
- Support email magic link or OTP login using a managed auth service.
- The app should be easy to customize later.

Tech stack:
- Next.js 14+ with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Framer Motion for subtle premium motion
- Zustand or React Context for lightweight state where appropriate
- IndexedDB for local-first AI chat history
- Supabase for authentication, user profile storage, friend system, realtime messaging, and presence
- Optional Prisma only if necessary, otherwise prefer Supabase directly for simplicity
- PWA support with manifest, icons, and offline-friendly shell
- Recharts for dashboards and analytics
- Lucide icons
